HTML, CSS, and JS. And to help the beginners in programming to gain more knowledge and understanding on sorting algorigthms by visualizing them.

This application supports the following sorting algorithms:

1. **Bubble Sort**.
2. **Insertion Sort**.
3. **Selection Sort**.
4. **Merge Sort**.
5. **Quick Sort**.
